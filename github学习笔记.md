## 学习内容

### GitHub的作用
GitHub 是一个基于 Git 的 **代码托管平台**，开发者可以在上面：
- 管理代码版本
- 进行团队协作
- 发布开源项目
- 构建个人作品集（Portfolio）


### GitHub的入门操作

#### 创建新仓库

1. 创建新仓库  
2. 填写项目名称、描述  
3. 选择公开（Public）或私有（Private）  
4. 创建新文件或上传文件  

#### git基本命令
1. git pull:从远程仓库拉取最新代码，并与本地分支合并。  
2. git push:将本地分支的最新提交推送到远程仓库。  
3. git commit:将暂存区（git add 添加的内容）提交到本地仓库。  
4. git add:把修改的文件添加到“暂存区”，为提交（commit）做准备。  
5. git clone:从远程仓库复制整个项目到本地。
