# 页面编后感
## 一
我设想做一个个人主页的网页，希望通过简单的界面展示文字、图片和动画效果。最后，我基本实现了原来的构想，网页可以完整展示内容，并且加入了一点点的交互效果，但是并未实现我预想的动画效果。
## 二
- 在项目中，我用 HTML 搭建网页的骨骼，包括头部标题、正文内容区、分区标题和段落。
- CSS 在项目中为网页增添了颜值。我使用了渐变背景、透明半透的内容区、圆角和阴影效果，使网页看起来现代且有层次感。
- 我用 JavaScript 给网页增加了动态变化，如点击显示
## 三
在项目中，我遇到的最大难题是内容区在屏幕上的布局尺寸的调整，和在不同设备上的显示不同，尤其是移动端显示不友好。但是由于时间有限，我最终并未完全解决移动端显示的问题。
## 四
当然需要优化，比如增加更多交互效果、优化移动端适配。如果将来我成为一名高手，我肯定希望回到最初为这个项目填上最后一块砖，让梦开始的地方变得更美好。
